This plugin will extend your parish website.

== Description ==

With this plugin you extend your parish website. With lot’s of cool features.
This plugin is designed to use in parish websites. You can use Kerk en IT Parish for the target audience you want. Either serving your parishioners or evangelize with the data where seekers are searching for.

This plugin will include:

*   Widget with the times of the masses in your parish. This is also available as shortcode.
*   An administration backend for fill in all necessary data witch are included in the widgets and shortcodes.

== Installation ==

1. From WP admin > Plugins > Add New
1. Search “Kerk en IT Parish” under search and hit Enter
1. Click "Install Now"
1. Click the Activate Plugin link

== Frequently Asked Questions ==

= Do I have to fill in duplicate data? =
No, this plugin will minimize the filled data. It’s all connect to each other.

= Why is this plugin is not suitable for our parish? =
You're expecting something from this plugin what is not in there. Please [contact us](http://www.kerkenit.nl/contact) to help us improving this plugin.
We will change the plugin to your requests when it's a good idea. We have some idea's to improve this plugin. But your's idea would be great.

= Do you have a roadmap? =
We've got some great coming up. Please [provide us feedback](http://www.kerkenit.nl/contact) for any request to extend this plugin.

= I miss data of my language? =
Currently this plugin will work perfectly in the Netherlands. We will add support for your language / country. [But we need YOUR help.](http://www.kerkenit.nl/contact)

= How can I contribute? =
You can help us by providing data of your local dioceses.
Also we are looking for translators. Please [contact us](http://www.kerkenit.nl/contact), and we will add support for your country.
